エイムボットはバックグラウンドで動作し、照準を制御して敵の頭部と胴体を瞬時に自動的に捕捉します。エイムボットの主な目的は、命中精度とキル数を向上させ、何時間もファームすることなく戦闘に勝利し、建築物を構築できるようにすることです。
